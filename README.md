Spring MVC bootstrap site
==========================

This project is Spring MVC + Bootstrap + Tiles application + Hibernate

It contains some useful configuration items:

- Spring 4.1.9
- Spring MVC
- Apache Tiles 3.0.5
- Bootstrap 3.3.6
- JQuery 1.11.0
- Hibernate 4.3.6
